This repository contains all code and design files for the TIIPS senior design project by Alexander Rens and Nile Morecraft, completed May 2017.
located at: https://github.com/alexanderprens/tiips_hardware

The folders contain:
	android app - .APK file for installing the TIIPS app
	barcodes - completed barcode designs and qr codes
	bluetooth code - code for the SiLabs BGM111 modules
	case files - STL files for the case
	pcb files - files for the PCB designs

Another repository contains the code for the Android application
located at: https://github.com/alexanderprens/tiips_android
